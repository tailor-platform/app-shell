## With Objects

<div style={{ maxWidth: 320, padding: "16px 0" }}>
  <Autocomplete
    items={[
      { code: "US", name: "United States" },
      { code: "JP", name: "Japan" },
      { code: "GB", name: "United Kingdom" },
      { code: "DE", name: "Germany" },
      { code: "FR", name: "France" },
    ]}
    mapItem={(item) => ({ label: item.name, key: item.code })}
    placeholder="Search countries..."
  />
</div>

```tsx
<Autocomplete
  items={[
    { code: "US", name: "United States" },
    { code: "JP", name: "Japan" },
    { code: "GB", name: "United Kingdom" },
    { code: "DE", name: "Germany" },
    { code: "FR", name: "France" },
  ]}
  mapItem={(item) => ({ label: item.name, key: item.code })}
  placeholder="Search countries..."
/>
```
