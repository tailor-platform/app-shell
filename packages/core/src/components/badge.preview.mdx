import { Badge } from "@tailor-platform/app-shell";

Use `Badge` to display short status indicators, labels, or tags — for example order status ("Shipped", "Pending"), priority levels, or category tags. Choose filled variants for high emphasis and outline variants with a status dot for subtler presentation.

## Filled Variants

<div style={{ display: "flex", gap: 8, flexWrap: "wrap", padding: "16px 0" }}>
  <Badge variant="default">Default</Badge>
  <Badge variant="success">Success</Badge>
  <Badge variant="warning">Warning</Badge>
  <Badge variant="error">Error</Badge>
  <Badge variant="neutral">Neutral</Badge>
</div>

```tsx
<Badge variant="default">Default</Badge>
<Badge variant="success">Success</Badge>
<Badge variant="warning">Warning</Badge>
<Badge variant="error">Error</Badge>

```

## Outline Variants

Outline variants include a colored status dot.

  <Badge variant="outline-error">Error</Badge>
  <Badge variant="outline-info">Info</Badge>
  <Badge variant="outline-neutral">Neutral</Badge>
</div>

```tsx
<Badge variant="outline-success">Success</Badge>
<Badge variant="outline-warning">Warning</Badge>
```
