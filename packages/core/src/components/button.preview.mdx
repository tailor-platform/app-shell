import { Button } from "@tailor-platform/app-shell";

# Button

A styled button with multiple visual variants and sizes. Supports rendering as a custom element via the `render` prop.

## Variants

<div style={{ display: "flex", gap: 8, flexWrap: "wrap", padding: "16px 0" }}>
  <Button variant="default">Default</Button>
  <Button variant="destructive">Destructive</Button>
  <Button variant="outline">Outline</Button>
  <Button variant="secondary">Secondary</Button>
  <Button variant="ghost">Ghost</Button>
  <Button variant="link">Link</Button>
</div>

```tsx
<Button variant="default">Default</Button>
<Button variant="destructive">Destructive</Button>
<Button variant="outline">Outline</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="ghost">Ghost</Button>
<Button variant="link">Link</Button>
```
