import { Button } from "@tailor-platform/app-shell";

Use `Button` for any clickable action — form submissions, confirmations, navigation triggers, and toolbar actions. Choose a `variant` to convey intent (e.g. `destructive` for deletions, `outline` for secondary actions) and a `size` to fit the surrounding layout.
