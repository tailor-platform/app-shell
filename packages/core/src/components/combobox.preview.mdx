## With Objects

<div style={{ maxWidth: 320, padding: "16px 0" }}>
  <Combobox
    items={[
      { id: 1, name: "React" },
      { id: 2, name: "Vue" },
      { id: 3, name: "Angular" },
      { id: 4, name: "Svelte" },
    ]}
    mapItem={(item) => ({ label: item.name, key: String(item.id) })}
    placeholder="Search frameworks..."
  />
</div>

```tsx
<Combobox
  items={[
    { id: 1, name: "React" },
    { id: 2, name: "Vue" },
    { id: 3, name: "Angular" },
    { id: 4, name: "Svelte" },
  ]}
  mapItem={(item) => ({ label: item.name, key: String(item.id) })}
  placeholder="Search frameworks..."
/>
```
