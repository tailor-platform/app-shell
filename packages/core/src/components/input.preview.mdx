import { Input } from "@tailor-platform/app-shell";

Use `Input` for single-line text entry — names, emails, passwords, search queries, and other short-form values. It wraps a native `<input>` with consistent theming and supports all standard HTML input attributes.

## Basic

<div style={{ display: "flex", flexDirection: "column", gap: 8, maxWidth: 320, padding: "16px 0" }}>
  <Input type="text" placeholder="Enter your name" />
  <Input type="email" placeholder="Email address" />
  <Input type="password" placeholder="Password" />
</div>

```tsx
<Input type="text" placeholder="Enter your name" />
<Input type="email" placeholder="Email address" />
<Input type="password" placeholder="Password" />
```
