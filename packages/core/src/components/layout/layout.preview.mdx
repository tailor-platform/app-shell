import { Layout } from "@tailor-platform/app-shell";

Use `Layout` to organize page content into 1, 2, or 3 responsive columns with an optional page header. Ideal for detail pages, dashboards, and forms where you need a consistent column grid that adapts to the viewport.

## Single Column

The simplest layout with a single full-width column.

<div style={{ padding: 16, border: "1px dashed var(--border)", borderRadius: 8 }}>
  <Layout>
    <Layout.Header title="Page Title" />
    <Layout.Column>
      <div style={{ background: "rgba(161, 161, 170, 0.2)", borderRadius: 6, padding: 16, textAlign: "center" }}>
        Main Content
      </div>
    </Layout.Column>
  </Layout>
</div>

<br />

```tsx
<Layout>
  <Layout.Header title="Page Title" />
  <Layout.Column>
    <p>Main content goes here.</p>
  </Layout.Column>
</Layout>
```

## With Header Actions

Use `actions` to place buttons alongside the page title.

<div style={{ padding: 16, border: "1px dashed var(--border)", borderRadius: 8 }}>
  <Layout>
    <Layout.Header
      title="Orders"
      actions={[
        <button
          key="export"
          style={{
            padding: "4px 12px",
            borderRadius: 6,
            border: "1px solid var(--border)",
            background: "var(--card-bg, transparent)",
            color: "var(--fg, inherit)",
            cursor: "pointer",
          }}
        >
          Export
        </button>,
        <button
          key="create"
          style={{
            padding: "4px 12px",
            borderRadius: 6,
            border: "none",
            background: "var(--fg, #18181b)",
            color: "var(--bg, white)",
            cursor: "pointer",
          }}
        >
          + New Order
        </button>,
      ]}
    />
    <Layout.Column>
      <div style={{ background: "rgba(161, 161, 170, 0.2)", borderRadius: 6, padding: 16, textAlign: "center" }}>
        Content area
      </div>
    </Layout.Column>
  </Layout>
</div>

<br />

```tsx
<Layout>
  <Layout.Header
    title="Orders"
    actions={[
      <Button key="export" variant="outline">
        Export
      </Button>,
      <Button key="create">+ New Order</Button>,
    ]}
  />
  <Layout.Column>
    <p>Content area</p>
  </Layout.Column>
</Layout>
```

## Area-Based Columns

Use the `area` prop to assign role-based widths (`left`, `main`, `right`) instead of relying on position.

<div style={{ padding: 16, border: "1px dashed var(--border)", borderRadius: 8 }}>
  <Layout>
    <Layout.Header title="Area-Based" />
    <Layout.Column area="left">
      <div style={{ background: "rgba(74, 222, 128, 0.2)", borderRadius: 6, padding: 16, textAlign: "center" }}>
        Left
      </div>
    </Layout.Column>
    <Layout.Column area="main">
      <div style={{ background: "rgba(96, 165, 250, 0.2)", borderRadius: 6, padding: 16, textAlign: "center" }}>
        Main
      </div>
    </Layout.Column>
    <Layout.Column area="right">
      <div style={{ background: "rgba(250, 204, 21, 0.2)", borderRadius: 6, padding: 16, textAlign: "center" }}>
        Right
      </div>
    </Layout.Column>
  </Layout>
</div>

<br />

```tsx
<Layout>
  <Layout.Header title="Area-Based" />
  <Layout.Column area="left">Left sidebar</Layout.Column>
  <Layout.Column area="main">Main content</Layout.Column>
  <Layout.Column area="right">Right sidebar</Layout.Column>
</Layout>
```

## Four Columns

        Column 1
      </div>
    </Layout.Column>
    <Layout.Column>
      <div style={{ background: "rgba(96, 165, 250, 0.2)", borderRadius: 6, padding: 16, textAlign: "center" }}>

</div>
