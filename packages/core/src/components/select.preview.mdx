## With Objects

<div style={{ maxWidth: 320, padding: "16px 0" }}>
  <Select
    items={[
      { id: 1, name: "Active" },
      { id: 2, name: "Inactive" },
      { id: 3, name: "Pending" },
    ]}
    mapItem={(item) => ({ label: item.name, key: String(item.id) })}
    placeholder="Select a status"
  />
</div>

```tsx
<Select
  items={[
    { id: 1, name: "Active" },
    { id: 2, name: "Inactive" },
    { id: 3, name: "Pending" },
  ]}
  mapItem={(item) => ({ label: item.name, key: String(item.id) })}
  placeholder="Select a status"
/>
```
