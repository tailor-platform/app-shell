import { Table } from "@tailor-platform/app-shell";

Use `Table` to display structured data in rows and columns — such as user lists, order histories, or inventory records. It provides pre-styled semantic HTML table sub-components wrapped in a horizontally scrollable container.

## Basic

```tsx preview
<div style={{ backgroundColor: "#ffffff", border: "1px solid #e0e0e0", borderRadius: 4, width: "100%" }}>
  <Table.Root>
    <Table.Header>
      <Table.Row>
        <Table.Head>Name</Table.Head>
        <Table.Head>Status</Table.Head>
        <Table.Head>Role</Table.Head>
      </Table.Row>
    </Table.Header>
    <Table.Body>
      <Table.Row>
        <Table.Cell>Alice</Table.Cell>
        <Table.Cell>Active</Table.Cell>
        <Table.Cell>Admin</Table.Cell>
      </Table.Row>
      <Table.Row>
        <Table.Cell>Bob</Table.Cell>
        <Table.Cell>Inactive</Table.Cell>
        <Table.Cell>Editor</Table.Cell>
      </Table.Row>
      <Table.Row>
        <Table.Cell>Charlie</Table.Cell>
        <Table.Cell>Active</Table.Cell>
        <Table.Cell>Viewer</Table.Cell>
      </Table.Row>
    </Table.Body>
  </Table.Root>
</div>
```
